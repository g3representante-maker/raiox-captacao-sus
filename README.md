# Raio-X de Captação SUS — G3 Health Service

Dashboard de diagnóstico de captação de recursos federais do SUS (custeio MAC e PAP)
por município, cruzando propostas (Consulta FNS) com descontos FAF (Portal FNS).

- 5.571 municípios · 27 UFs · exercícios 2024–2026
- Ranking por número único / recuperável / por habitante
- Diagnóstico por município, proposta de trabalho em PDF e envio por WhatsApp

Acesse: `index.html` (GitHub Pages).
